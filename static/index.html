        用户上传原始财务数据（科目余额表、明细账、对账单等）后，系统通过 openpyxl 解析 Excel 文件，
        将数据序列化为结构化文本。随后，系统加载预置的审计准则（CLAUDE.md），与用户数据组装为 Prompt，
        通过 DeepSeek API（兼容 Anthropic Messages 协议）调用大语言模型。
        LLM 依据中国注册会计师审计准则执行分析性复核、函证检查、账龄分析等程序，返回结构化审计数据。
        服务端 Python 模板引擎将数据渲染为排版规范的 Excel 底稿，同时输出 Markdown 格式的审计日志。
