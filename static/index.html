用户上传原始财务数据后，系统通过 openpyxl 解析 Excel，序列化为结构化文本。加载预置审计准则（CLAUDE.md），组装 Prompt 通过 DeepSeek API 调用大语言模型。LLM 依据中国注册会计师审计准则执行分析性复核、函证检查等程序，返回结构化审计数据。服务端 Python 模板引擎渲染为排版规范的 Excel 底稿，同时输出 Markdown 审计日志。
